さようならVAIO Type TZ

VAIO type TZを購入して一年も経っていないのにMacBook Airを購入し使い始めてまだ一ヶ月になっていない今日この頃、そのVAIO type TZを手放しました。

何台もパソコンを持っていても使う人は一人なわけで、メインであるMacBook Airを毎日使っているとその分VAIO type TZを使う頻度が減ってきます。ましてや、ThinkPad X40やMac miniやら気がつくと手元には少なからずマシンがあったりと・・・。そんな中でこのVAIO type TZはまだ新しいですし現役で十分に使える性能でもあります。使う頻度が減った私のところで眠らせておくよりは使ってくれる人がいるのであればその人の元へ出す方が、マシンにとっても良いことではないでしょうか。

ということで、同僚が買いたいと手を挙げてくれたので思い切って売りました。実は、自分のマシンを他の人に売るということは、今回が初めてです。今までは自分で買ったものは絶対に売らず手元に残すか親に無償提供ということをしていたのですが、初めてです。今思うとちょっと寂しい気持ちもあるような気がしないでもないですが、ね、やはり使ってくれる方のところで使ってもらえる方がVAIO type TZも嬉しいハズですし、これで良かったと思っています。

さようならVAIO type TZ。そして、こんにちはMacBook Air。

新宿御苑

春麗らかな陽気になった今日、新宿御苑に行ってきました。

いつも通りゲートを潜って、ふと芝生のほうに目をやると凄かったです。花見の季節の頃のように、人、人、人という感じで大勢の人がいました。ついこの間までは週末でもそんなに人はいなかったのですが、やはり今日のように暖かくなるとみんな出てくるわけですね。春です、春です。

寒桜に梅に、今がちょうど見頃ではないでしょうか。それらの花を見ながらお弁当を食す、何と素晴らしいことでしょうか、ワクワクウキウキしてきます。しかも今日は綺麗な青空が広がる良い天気ということもあり、写真を撮りまくりました。やはり春は良いですね。

α350撮影体験セミナー

私が使っているデジタル一眼レフカメラはNikonのD80なわけなんですが、ちょっと興味があるというかお触り会にいってみようよという話になり申し込んでいた「かんたんきれいに毎日残そう α350 撮影体験セミナー」にkinaさんとTAKESAKOさんとruriさんとで行ってきました。

内容的には、講師の吉住志穂先生がα350の機能説明等々をしてくれて、その後、各自、モデルさんやアイボやお花をα350を使って撮影という流れでした。レンズもいろいろ準備されていて自由に交換して撮影をすることができたりと、ものすごく楽しかったです。講師の吉住志穂先生もいい感じの人で撮影した作品も素敵なものでとても良かったです。α350の感想はというと、ファインダの見やすさはやはりD80が一番だなと思いつつ、あのライブビューは欲しい・・・。私の場合、写真を撮るときよく地面スレスレというか結構無理な体勢をして花を撮ることが多々あるので、楽な姿勢で撮ることができるこのライブビュー凄く良い！羨ましいです。

うーん、いろいろ触って撮ったらα350欲しいなぁ。しかし、D80あるしレンズはもちろん使い回せませんし、困った困った。

何はともあれとにかく楽しいセミナーでした。しかし、ただ一点、撮影したデータを持ち帰れないというのが残念でした。その場で何枚かは印刷してくれるのですが、やはり撮影したデータが欲しかったです。いろいろと事情はあるのでしょうが、ちょっと残念でした。

予約録画

テレ朝にて放送されている「ちい散歩」にて知っているお店が出るということを思い出して、会社についてすぐ携帯電話のワンセグで録画予約をしました。電波の状態などをチェックする余裕がなかったので、うまく撮れていることを祈りつつ、一日が終わり、うちに帰ってきて先ほど録画した番組を観てみたのですが、ちょうどそのお店のところで電波の状態が不安定だったようで、カクカク動きになったり映像が止まったりと、ちょっと残念な結果になりました。でもまあ、ある程度観られたのでそれはそれで良かったのですが、なんといいますか、電波状態を確認しておくんだったとちょっと反省です。

しかし、まあ観られたことには変わりないので、今度お店に行ったらあのおばちゃんに「観たよ！」っていおこうと思います。

カレー

有楽町イトシアでお昼ご飯を食しました。映画「ヒトラーの贋札」を観るのでというか、あと45分でその映画が始まるのでそれまでに食さなければなりません。そんなこともあり近くだった有楽町イトシアにてその中にある東京カレー屋名店会で食しました。

私が注文したのでAセットなるインド風ポークカリーと名店会オリジナルチキンカレーのセットです。食してみると意外と美味しいカレーで、なかでもインド風ポークカリーが辛い辛い。辛いものが平気な私ではあるのですが、正直辛かったです。なんか、久しぶりにこんなに辛いものを食しました。大満足です！

ちなみに、この後観た「ヒトラーの贋札」ですが結構面白かったです。
