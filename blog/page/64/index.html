スパムに認定？


MNPでさんざんキャリアを渡り歩き、電話番号はそのままだけどメールアドレスが変わりまくっているので、いい加減面倒になってきました。そんなこともありiPhone用にGmailのアカウントを取得し、これからはそれを使おうと思いお知らせメールを送信しているのですが・・・、送信しすぎなんでしょうか？エラーで送信できなくなることが多々あります。

See http://mail.google.com/support/bin/answer.py?answer=69585 for more information.

ということで、どうもGmailのほうでスパムに認定されてしまったらしく送信できないようです。ね！そんなに送りまくっているわけでもないし、BCCに6件追加したら引っかかってしまいました。いろいろググると24時間経てば解除されるらしいんですが・・・。で、そんなことがあった次の日、問題なく送信できるようになりました。がしかし、また送信できなくなりました。前回のことを踏まえてBCCは使ってないし、普通にTOで一人一人にちまちまと送ったのに！うむー、うむー、Gmail使えないなぁ。こんなことが続くようなら、お知らせメール送っている最中だけど、別なものに変えようかなと思い始めている今日この頃です。。。。

iPhone 3G


なんだかんだいって、実は昨日iPhone 3Gを購入しました。16GBのホワイトです。しかもまた懲りずにMNPでdocomoからSoftBankに移っています。これでdocomo、au、SoftBankと3キャリアを渡り歩いています。うむーうむー。でもって、MNPで移ったということで電話番号は同じなんですがメールアドレスが変わります。で、もういい加減面倒なのでこの際Gmailにしようかと思うのですが、諸事情でまだ設定できてません。以前の携帯のメールアドレスはもうすでに使えなくなっているのですが、このような状態でメールアドレスの設定が終わっていないので、まだ皆さんにお知らせできていません。明日・明後日中にはお知らせできるかと思いますので、よろしくお願いします。

でもって、iPhone 3Gなんですが、ぶっちゃけメール関係の部分だけでいえば、日本の携帯電話というかiPhoneを除くスマートフォーンを含めてというか、それらの方が実用的な感じがします。まあiPhoneを購入する前にその辺のことはわかっていたのですが、実際に手にして使ってみてよく実感できたというか何というか。いわゆるプッシュとか着信ランプ等々、メールを中心としたやりとりにはこのiPhoneは向かない感じがしました。

ここだけの話、MNPで移ってメインの携帯にするよりは、新規で新たに電話番号を取得して電話も出来るiPodとして使う方が良かったかもしれません。でもまあ使っていけば思いも変わるのかもしれませんが。

その他の機能としては今までiPod touchを使っていたということもあり特に変わりはないかなという感じです。あ、あともう一ついいたいことが、電波がすっげー入りにくいです。

ちなみに、モバイルSuicaやEdyを使いたいのでdocomoを新たに契約しています。ということで、今現在SoftBankとdocomoとWillcomの三台持ちという環境になっています。ほぇー。

北海道旅行 最終日


あっという間に最終日になった北海道旅行。今日は昨日五稜郭でたまたま見た張り紙に書かれてあった市立函館博物館で開催されている特別展、没後100年「榎本武揚−箱館戦争の光と影−」を観に行ってきました。幕末好き・榎本武揚好きとしては函館に来ているのだからこれは観なくてはいけません。

市電に揺られて博物館のあるところまで移動。ちょっと雨がちらついたりしてきた感じなんですが、何とか濡れる前に博物館に入ることが出来ました。内容的にはすべて榎本武揚！ということではなくてちょっと私の想像とは違っていたのですが、それはそれでなかなか楽しい内容でした。

この後はちょっと距離があってどうしようか迷っていた外人墓地に行ったりしました。何といいますか、あっという間に北海道旅行が終わってしまいましたよ。そんなことで19時近くの飛行機で東京に帰ってきました。

とても楽しい旅行になった今回の北海道旅行、また行きたいです！

北海道旅行 2日目


そんなことで二日目。北海道に来たからにはやはり新鮮な海鮮を食しませんとね！そんなことで朝食はRyuさん一家の案内で漁師の直売店へ行き海鮮を食しました。私が注文したのはウニ丼。このウニがまたとても美味しくて、ウニが苦手な人でもいけるんじゃね？と思えるほど凄く美味しいウニでした。 で、そろそろ駅に向かわないといけないという時間になったわけですが、ケイさんおすすめの途中にあるお菓子屋さんで「岩シュー」を買い食い。注文してからその場で中にクリームを詰め込むというなんかこだわりのシュークリーム、しかもこのサクサク感を味わうため賞味期限は1時間という代物です。味はというと、その文言通り外はサクサク中はクリームのしっとり感、この絶妙さが堪らないとても美味しいシュークリームでした。

そんなことで、Ryuさん一家にはとてもとてもお世話になったひととき。ありがとうございました。

後ろ髪引かれつつもお昼近くの電車で函館まで戻ってきました。ちなみに昼食はこれまたおすすめのカニ飯の駅弁を食しました。カニがたっぷりのこの駅弁これまた美味しかったです。で、一つ目の目的を達成した次は、もう一つの目的である五稜郭です。

私にとって五稜郭とは、子供の頃観た時代劇や幕末物の話でよく目にし耳にし、観てみたい行ってみたいと思い続けていた場所です。そんな五稜郭が目の前にあるわけでこれは興奮するしかありません。五稜郭タワーに上り上から眺める五稜郭、素晴らしいですね、美しいですね。これは感動ものです。幕末にこの五稜郭で榎本武揚が・・・とか考えると堪りませんね。もちろん上から眺めるだけでなく、五稜郭に入りその地も踏んできました。もちろん、五稜郭の中と行っても今は公園で特に特別なものがあるわけではないのですが、感無量です。

五稜郭を満喫した後は、近くにあるラーメン屋さん「あじさい」で塩ラーメンを食しました。ラーメンといったら私の中では味噌ラーメンが一番好きなんですが、ここの塩ラーメンを食してみたら、なんか変わりました。塩ラーメンって美味しいんですね！あっさりしていながらもしっかりした味、これはハマります。

五稜郭と塩ラーメンを満喫した後は、幕末から明治にかけて建てられた洋館を観て大満足し、函館山へロープウェイで登ったものの天気が悪く霧で真っ白で何にも見えずにがっかりしたり、夕食はお寿司屋さんでお腹いっぱいになったりと、ものすごく充実した一日でありました！

ちなみに、東京に住んでいながら今までいわゆる芸能人を生で見たことがなかったのですが、五稜郭タワーで元野球選手で元大リーガーの某芸能人を見ました。芸能人もいろいろ大変なんですね。

北海道旅行 1日目


以前から今年の夏は北海道へ行くよと話し合っていました。で、どうも今年の夏休みは仕事が忙しくて「夏」にはとれそうにもないので、じゃあ今のうちに行こうかと言うことになり計画していた北海道函館旅行、ついにこの日がやってきました。朝10時過ぎの飛行機で函館へ出発です。

今回のこの旅行の大きな目的は二つあります。一つは北海道に住むRyuさん・ケイさん・アオイくんに会いに行く、もう一つは子供の頃から憧れていた五稜郭を観るということです。

そんなことで、まず初日の今日は一つ目の目的であるRyuさん一家に会うという予定で行動しました。

函館空港に着いたのは12時ちょっと過ぎ。まずはということで駅近くの「どんぶり横丁市場」でイカ・いくら丼を食しました。函館に来たからにはやはりイカですよね！美味しいイカに満足しお腹いっぱいになったところで、さっそく電車に乗り移動しました。

揺られること1時間ぐらい、駅まで迎えにきてくれたRyuさん一家と久しぶりにご対面。アオイくんとはこれが初対面です。以前Ryuさん・ケイさんと東京でお会いしたときはまだケイさんのお腹にいたアオイくんがこんなにも大きくなって、しかもこれまた可愛いー。でもってRyuさん案内の元、美味しい水がわき出てるところや、美味しいとろけるようなアイスクリームを食したり、これこそ北海道という広大な素晴らしい眺めを堪能したり、サフォークなるこれまた凄く美味しい羊肉をバーベキューで食したりと、北海道を満喫した一日となりました。夜は夜で、お宅にお邪魔して噂のポンタくんとご対面したり、おしゃべりしたり、楽しいひとときを過ごすことが出来ました。ありがとうございます！

銀座ソニービル


先日発表された新しいVAIO type Zの実機を触りに銀座ソニービルに行ってきました。

さすがソニーというか何というかいやに外国の方が大勢見に来ていた今日のソニービル、私はというとtype Zは何処ですかーという感じで、もうそれしか頭にないような状態であちこち探し回っていました。で、早速発見したtype Zの実機。残念なことに1600x900という解像度の液晶を持つものではなく、普通というか1366x768のもでした。でもまあサワサワと触りまくったわけなんですが、見た目から想像する重さとは違い結構軽かったり、また思いっきり負荷をかけたりしたわけではないのであれですが普通に使っている分には熱さは気にならず、で、GPUを切り替えたりしてVistaのWindowsエクスペリエンスインデックスのスコアを更新してみたりしたら、ファンから結構熱風が吹き出してました。まあ当然のことかもしれませんが。。。

しかししかし、結構良い感じのものでした。まあ。ぶっちゃけ今使用しているMacBook Airよりいろんな意味で実用的だなと思いました！でも、質感的な目線で見るとMacBook Airの方が上という感じはしました。iPhone 3Gは手に入らないしさ、type Z欲しいな！

ATOK 2008 for Mac


私にとって日本語入力ソフトであるATOKは必要不可欠なものです。もちろん漢字読むことが出来ますし書くことも出来ますが、でもやはりATOKがないと不安で不安で満足に文章を書くことが出来ません。そんなATOKに依存しまくりの私ですが、今日は待ちに待ったATOK 2008 for Macの発売日です。

MacBook Airを購入してすぐにATOK 2007を購入したわけなんですが、アップデートモジュールをインストールしたとしてもLeopardにしっかりと対応しているわけではないので、なんか動作が不安定というか、ATOKらしくないというか、Windowsと比較しても何かが違いました。そんなことで、Leopardにちゃんと対応したバージョンを待ち望んでいたわけなんですが、ついにこの日がやってきました！

さっそくJust MyShopにてダウンロード販売をポチッと。そして、つい今し方インストールしました。で、感想はというと2007とは違って良い感じです。今までのが明らかにおかしかったというかATOKらしくない日本語変換というか、ユーザ辞書の初期化をしてもなんか違っていたんです。Windowsで使っていた頃のようなあの快適な日本語変換が出来ないでいました。しかししかし、この2008は望み通りというかこれでこそATOKという感じで、長文の日本語変換も快適です。Windowsで使っていた頃のような快適さを感じることが出来ます。すばらしい。やはりATOKはこうでなくちゃ！
