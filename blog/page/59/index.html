ナマステ・インディア

お昼ごはん


先日行った讃岐うどんのお店「元喜」へまた行ってきましたよ。

今回は開店直後ということもあり並ぶことなく席に着くことができました。しかし、席についてふと入口を見たところ人がぞろぞろと入ってきて、あっという間に満席、注文し終えるともうすでに外には行列ができていて、ほんとタッチの差ですんなりと中に入ることができて良かったです。

ちなみに、今回私が注文したのはエビ天ぶっかけです。エビのてんぷらをメインに野菜のてんぷらもちょっとのっているぶっかけうどん。このコシがあるうどんにカラッと揚がったてんぷらがこれまた美味しくて、私は大満足です。

相変わらずおなかいっぱいになり、美味しいし、この「元喜」私のお気に入りなりましたよ！全メニュー制覇したいです。

肉みそうどん


お昼はどうする？という話になり、以前から今度行ってみようよと言っていた近くにある讃岐うどんのお店「元喜」に行ってみました。

近くということもありてくてくと歩いて行ってみたのですが、お店の前には結構行列ができていました。近所にこんな感じの行列のお店があるとは思いもしませんでしたよ。折角来たのだからということで並んでうどんを食してきたのですが、結論を言えば、大満足とても美味しかったです！

私が注文したのは夏季限定の肉みそうどんなんですが、これがまた素晴らしく美味しかった！エビ餅ぶっかけも口にしてみたのですがこちらもプリプリモチモチしていてすごい美味しかったです。で、人のものが美味しく見えるという言葉があるように、遠目で彼方此方の人のものを見ていたら天ぷら系がこれまた美味しく見えまして、次回はあの天ぷらものを注文する！と心に決めた私です。

そんなことで、近々またここで食したいです。

VAIO TTかThinkPad X200sか

VAIO type TZの新型を購入するまでのつなぎとしてThinkPad X61sを購入し、日々、Windows Vistaの快適さを実感している今日この頃。そのtype TZの新型のスペックが出てきました。

TTですよTT。VAIO type TTになるのかな。TTというと車、アウディが思い浮かぶわけですが、とにかくTTです。スペック的にはtype ZからGPUを抜いて小さくした感じ、外見的には今のtype TZとどこが変わったのかあまりよくわからない感じに見えます。しかし値段がいやに高いです、この辺はもちろんカスタマイズできるとは思うのですが。。。私的にはチップセットが現行世代になりメモリーMAX4GBになったというのは大変喜ばしいです。デザインもほぼ変わらない？ようなので、これは買いかな。

しかししかし、そんなことを思っているところに今度はThinkPad X200sの販売がアメリカでは始まったようで、そのスペックがまたちょっと惹かれます。12.1インチの液晶画面でWXGA+ 、1440x900の解像度ですよ！いいな。

なんだかものすごく迷ってきました。VAIO TTとThinkPad X200s。比較すると大きさ的には279x199.8mmと295x210mmでもちろんTTのほうが小さいわけですが、20mmほどということでそれほど大きな差はないような気がします。それでいて重さは1.3kgと1.12kgということでX200sがちょっと軽い。そして解像度が1366×768と1440x900。おおおー、やばい悩みます。ただ番号が3桁になったThinkPad（もしかしたら今のところはX200シリーズだけ？）の私的欠点として、天板の隅にLenovoと彫りこまれていることです。今まで通りThinkPadのロゴだけであればよかったのですが反対の隅にメーカーロゴまで彫りこまなくてもと思うのですが。

明日になるとVAIO TTのもっと詳しい情報が出てくるのかな。ThinkPad X200sも早く日本で発表ならないかな。VAIO TTにするかThinkPad X200sにするか、それともVAIO type Zにするか、ううう、悩みます！

おくりびと


ベトナムフェスティバルの帰り、映画「おくりびと」を観てきました。

ロケ地が山形で音楽が私の好きな久石譲ということもあり、予告等々で目にするたびこれは観たいと思っていました。しかも何といっても泣けそうな映画であるということがさらに興味を引きました。というのも、予告を見てすでに私泣いています。

そんなことで「おくりびと」ですが、結論からいえば私泣きまくっていました。涙って本当にしょっぱいんですね。それが実感できるくらい泣いていました。やばかったです。とても素晴らしい内容の映画でもう一回観てもまた泣けると思う映画であり、観て本当によかったと思える内容でした。

ちなみに、そのあと「大庵」で晩御飯を食したのですが、このお店、素晴らしく美味しかったです！秋刀魚の刺身に出汁巻き玉子にそしてお蕎麦、お酒によく合ってツルツルって感じでとても美味しく食すことができました。

今日を振り返ってみると、美味しいものをおなかいっぱい食すことができ泣ける映画を観ることができ、充実した一日でありました。
