MacBookかVAIOか


ついこの間までMacBook Airを使用していながらも、結局何だかんだの問題でメインで使用することをあきらめThinkPad X61sを購入しWindows Vistaに満足していた私なんですが、今日になり新しいMacBookが欲しくなりました。えへ。

というのも、会社の同僚がその新しいMacBookを購入し今日持ってきたわけなんですが、実物を目にし触れてみたりしたところ、格好いいんですよ。MacBook Airで懲りたというのになんなんでしょうね。実物を目にすると駄目ですね。確かに重くて毎日持ち運ぶのはちょっと辛いかなとか思ったりもするんですが、その質感というかデザインはさすがアップルといった感じです。今使用しているThinkPad X61sは以前のIBM時代のThinkPadとは違いパームレスト部分などちょっと安っぽさがあり（まあ事実安かったので仕方のないところなんですが）所有する喜びというものはあまり感じられません。でも、アップルの製品は違います。使えるかどうかは別にしてもその高いデザイン性からも所有したときの喜びは格別なものがあります。

VAIOを購入しようかと思い続けながらも未だ検討中の私。もう一度、アップルというかMacBookに挑戦してみる？それとも素直にVAIO type Tにしてみる？はたまたVAIO type Zにしてみる？

ちなみに、現時点で私の希望するMacBookの構成では\205,420、VAIO type Tは\194,800ということで、ほぼ同じような値段。で、ちょっと離れてVAIO type Zは\251,800 です。うむーうむー悩みますな。

エスプレッソメーカー


コーヒー大好き人間の私。普段はその手軽さからインスタントコーヒーを飲んでいるのですが、なんか良さげなものがあるよということで、先日無印良品でアルミ製のエスプレッソメーカーを購入しました。

このエスプレッソメーカー、直火式ということでこれをそのまま火にかけることができます。下の部分に水を入れて、真ん中の部分にはコーヒー豆をセット。で、火にかけて沸騰すると上記がそのコーヒー豆がセットしてある部分を通り、上の部分に溜まるという仕組みになっています。

早速豆を買ってきて作ってみたのですが、楽しい！です。これはいいですね。エスプレッソということで、もちろん、いわゆるコーヒーカップに対して入れると量は少ないわけなんですが、おうちでこういうものが楽しめるというは素晴らしいです。いままでは先に述べたようにインスタントコーヒーということでコーヒーが飲めれば良いという感じだったのですが、エスプレッソメーカーを手にした今、コーヒー豆にもこだわりたいです！

ぱんだ珈琲店


ペンギンが大好きなんですが同じ白黒であるパンダも大好きな私。何もかもがパンダである「ぱんだ珈琲店」なるお店が阿佐ヶ谷にあるということを知り、行こうぜ行こうぜということで、kinaさんとruriさんとTAKESAKOさんとTsukimiyaさんとで行ってきました。

開店と同時にお店に入ったということもあり、もう一組小さなお子さんを連れた家族がいただけだったので、思う存分店内の写真を撮りまくってしまいました。彼方此方にパンダがいて、これはパンダ好きにはたまらないお店です。オムライスにはケチャップでパンダの絵が描かれていて、カフェオレにはココアパウダーでパンダが描かれ、クッキーはパンダ。素晴らしいです。どれもこれも食べるのがもったいないくらいで、しばらくの間、描かれているパンダに見とれていました。

今度は夜の部で是非ともハンバーグを食したいです。ハンバーグに描かれているパンダをみたいです。行こうぜ！

そんなことで、撮った写真はFlickrにアップロードしましたので、観てみてください。
