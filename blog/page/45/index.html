Windows 7 RC


先日次期WindowsであるWindows 7のRC版が公開されダウンロードできるようになったわけですが、もちろん私も早速ダウンロードしてVAIO type Pにインストールしてみました。

私のtype PにはXPをインストールして快適に使っています。せっかくここまで時間をかけて作った環境を壊すのは忍びなかったのですが、誘惑には勝てなかったというかなんというか、さくっとWindows 7 RCをクリーンインストールしちゃいました。

インストール自体30分かからないかどうかという感じの短時間で終了、あっという間でした。でもってWindows UpdateをかけたりするとディスプレイドライバであるGMA500のアップデートやいわゆるsony piのドライバだったりが入って、さらにリカバリーディスクからVAIO関係のソフトウエアを抽出してインストールしてセットアップ完了です。

使ってみた感想は、Vistaよりかなりさくさく動作するということです。XPと比較するとどちらかといえばXPかなという感じでしょうか。ただちょっとびっくりしたのはWindows 7でAeroをオンにしても普通に使えるということです。Vistaのときは使い物にならないぐらい重かったものがWindows 7では普通に使えます。見た目重視の私としてはAeroで普通に使えるのであればXPよりも断然いいです。それに、XPのときはFirefoxなりGoogle Chromeなりブラウザ内でスクロールしたときなどちょっと引っ掛かりがあったような気がしていたのですが、これまたWindows 7では気にならない感じで普通にスクロールできています。そんなことで、Windows 7はものすごく期待できるOSかも！とか、久しぶりに正式版リリースが楽しみなMicrosoftのOSかもしれません。

type PでWindows 7は意外と快適です。Aeroも使えるので半透明好きな私としてはXPからようやく卒業できそうです。

映画と晩ご飯


今日も鎌倉に行こうかと思ったりもしたのですが外はあいにくの雨ということで、映画を観に行ってきました。「スラムドッグ＄ミリオネア」です。公開から少なからず日が経っているし連休だし人は少ないだろうと思っていたのですが、そんなことはないほぼ満席でした。

クイズ＄ミリオネア自体はあまり好みじゃないので観てはいないのですが内容はよく知っています。スラム出身の少年がその番組でミリオネアになりチートしたのか否かなぜ答えを知っていたのか、その少年のこれまでの人生を振り返りつつ進むこの映画、内容にテンポに素晴らしいものでした。

でもって、晩ご飯は「とろさば鮨と穴子鮨」です。帰りに伊勢丹に寄ったら寿司とお茶の催し物をやっていてものすごく美味しそうだったので、たまには豪華なものもいいかなと。早速食してみると、脂がのっていて素晴らしく美味しいさば鮨、これは旨いです！

またまた鎌倉


土曜日に鎌倉に遊びに行ったばかりなわけですが、今日もまた早起きをして鎌倉に遊びに行ってきました。先日とは違い体調は万全、ワクワク気分で電車に揺られ出発です。

ゴールデンウィークも中頃ということもあり人がものすごく多い鎌倉。先日来たときは「ゴールデンウィークでもこのくらいの人出かこれなら全然大丈夫だな」と思っていたのですが、それは間違いでした。これが本当の姿だったのかというくらいどこを見ても人人人、ふんだんならこのお店は並んでいない（失礼ですが）というようなお店でも人が列をなしていました。すごいですね。そんなことで私たちはもちろん「OXTMORON」へ直行です。まだ開店まで10分かそこらという時間に到着することが出来、もちろん一番乗りで店内に入ることが出来ました。今回私が注文したのは「マトンカレー」です。マトン好きの私としては前々から注文したいと思っていたものの他のメニューに目移りしていてなかなか注文できずにいましたが、今日ようやく注文することが出来ました。マントと大根という組み合わせのこのカレー、やばいですね、ものすごく美味しいです！

おなかいっぱいになった後は、鶴岡八幡宮へ行き丁度ぼたん苑のぼたんが見頃ということで見てきました。大きく綺麗なぼたんが咲き誇っていて、また、それぞれのぼたんには雨よけ？という感じで傘が差してあり風情がでていてとても綺麗でした。その後はちょっとお寺巡りをしてみたものの、とにかくどこも人が多いということがあり今日は早めに引き上げることにしました。ゴールデンウィークはお出かけするのもなかなか大変です。でも、マトンカレーも食せたし綺麗なぼたんも観られたし今日もまた大満足な一日でありました。

鎌倉


連休というかいわゆるゴールデンウィークに突入した今日、日帰りで鎌倉へ遊びに行ってきました。鎌倉へ遊びに行くとなるといつもは1泊とか必ず泊まりがけで行っているわけなんですが、今回は諸事情により日帰りです。

朝起きたらちょっと気持ち悪いし頭痛はするしで、もしかしたら電車の中で吐いてしまうんじゃないかなという、そのくらい体調が悪い状態で本当に私はいけるのだろうか？とか思ったのですが、行くと決めていたからには行かないといけません。そんなことで普段は出来る限り飲まないようにしている薬などを飲みつつ、鎌倉に出発です。

電車に揺られること1時間とちょっと。寝ていたのが功を奏したのか、朝のあのものすごい体調から回復しちょっと頭痛は残るもののだいぶ楽になり、朝食を食べなかったということもありすっかりおなかが空いた状態で食欲も回復しました。ということで、今回の鎌倉に来た一番の目的である小町通りにあるお店「OXYMORON」でカレーを食しました。前回鎌倉に遊びに来たときからここのカレーにハマってしまいまた近々来なければいけないなと思っていました。今回私が注文したのは本日のカリーの「豚肉のココナッツカリー」です。スパイスがよく効きつつもココナッツの甘い上品な香りが広がりそして後からピリリとくる辛さ。ものすごく美味しいです！

おなかいっぱいになった後は、鎌倉に来たからにはこの季節これを食せねばということで鶴岡八幡宮の参道沿いにあるお漬け物屋さんで「アイス胡瓜」を食しました。ここによるたび「アイス茄子」とどちらにしようか迷うわけなんですが、前回は茄子を食したので今回は胡瓜にしました。割り箸に丸ごと一本胡瓜又は茄子のお漬け物が刺さっているこの食べ物、食べ応えがありお漬け物大好き人間には堪らないものですね、大満足美味しかったです！

ということで、この他に雑貨屋さんを巡ったりちょっとだけお寺巡りをしたり、とても充実した一日でありました。

続・iida G9


メインの携帯電話をiPhoneからiida G9に変更し使用してほぼ一週間になりました。docomoのSO905iも使用しているのであれですが、メインとしては久しぶりに使うことになる日本の携帯電話というか今風にいえばガラパゴス携帯ということになるのでしょうか。先日も触れたようにiPhoneはiPhoneで素晴らしいとは思っていますが、私にとっては日本の携帯電話が最強だと改めて実感しています。

ということでiida G9についてなんですが、デザインの面では見るからにiPhoneを意識していると思わせるような外見・パッケージになっています。デザイン携帯ということなので意識せざるを得ないのでしょう。 開けると黒のスポンジ？に収まって本体が収納されていたりします。無機質ないかにも家電的なパッケージというかとにかく収まっていればいい的なパッケージよりこういう方が所有したときの喜びはあるわけでなかなか良いです。しかし、今的にいえばというかエコな面でいえばどうせパッケージは捨てるしそこまで凝らなくても良いとか思ったりもしないでもないわけで、うーん人間ってわがままですね。

操作に関してですが、auの機種との比較という面であれば以前私が使用したことがあるのはペンギンのW51CAなんですが、あれと比較すると全然中身が違い比較にはならない感じです。すみません。あのとき、auの携帯電話ってdocomoのFOMAと比較するとタスク切り替えが使い物にならないなとか思っていました。で、それから時が経ち、最近のauはもっさりで使い物にならないとかそんな話を耳にしていたこともあり、ちょっと心配していました。がしかし、そんな心配は無用でサクサク動作しています。しかもタスク切り替えもいろんな機能の切り替えが出来るようになっていて、docomoのそれと同じくらい、いやdocomoはdocomoでというかSOに関してはでしょうかすぐにメモリー不足になりそんなにタスクを開いておくことは出来なかったのですが、このG9に関しては今のところそんな感じは全然ありません。

で、私にとって一番重要なものであるカメラですが、さすがに候補の一つであったS001と比較すればスペックは落ちていますが、携帯電話に付いているカメラとしてはごく普通ではないでしょうか。誕生日の写真はこのG9で撮りました。また、携帯百景に投稿している写真もG9です。

ちなみにこのG9で私的に気に入っているのはこのフォトビューワです。サクサク動作するし日付ごとにすらーっと写真が表示されるので意味もなくよく使っています。ちなみに、携帯電話を縦にしたり横にしたりするとちゃんとそれに追随して写真もちゃんと回転します（残念なことにガイドは横向きのままですが）。

そんなことでiida G9のレビューでした。やっぱり日本の携帯電話はいいですね！ソニエリ大好きです。

Iida G9


今メインで使用している携帯電話はiPhoneなんですが、ぶっちゃけメインとして使用するには不満があり日々文句たらたらな状態でした。Suicaを多用している私としては一度知ってしまったモバイルSuicaの便利さから離れることが出来ず、iPhoneとは別にdocomoも持ち歩いています。また、iPhoneでのメールの使用についてもいろいろ。

そんなことで、もうとにかく我慢が出来なかったので、タイトルと写真の通りMNPでauに移っちゃいました！！実はソニエリのS001にする気満々だったんですが、iida G9の実物を見て触ったらこちらの方が良い感じだったものでつい。ソニエリのあのマークは付いていませんがこのiida G9はソニエリ製で中身的にS001からカメラの画素数等々一部スペックを落としつつ外側を変えたものなので、私的には中身の面でも文句なしです。新しいメールアドレスについてはまだ端末の使い方や設定などいろいろ遊んでいる途中なので完了しましたらご連絡します。ちなみに以前auで使用していたものと同じのが取得できたのでそれにしてます。

やはり私的にはボタンというかキーというかハードウェアのキーが絶対必要です。また、iPhoneはメインよりサブとして使用する方が、通話やメールをメインとした携帯電話機というものではなく、ネットワークを利用した情報取得端末またはゲーム機というものとして使用する方が良いというのが私の結論になりました。あのときいうことを聞いてMNPでiPhoneに移らず新規で契約すべきだったと反省しています。ごめんなさい。すみません。これからはいうことをちゃんと聞きます＞＜
