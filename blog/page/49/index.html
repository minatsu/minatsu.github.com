VAIO Type P


いつになったら届くんだろうといっていた私のVAIO type Pですが、実は昨日の夕方届きました。そんなことで早速セットアップをしているんですが、今現在問題が一つ発生中です。それはリカバリDVDが作成できないこと。5回ほど試しているんですが半分ぐらいいったところで「作成できませんでした」とエラーメッセージが表示されて終了してしまいます。うむー。安かったということもありソニーのDVD-Rを購入してきたのですが、このメディアが悪かったのでしょうか。まあ今度は違うメディアを購入して試してみようかと思うのですが、USBの転送速度が追いつかないのでしょうか。うむー。

問題が一点解決できないでいるわけではありますが、type P小さくて軽くて素晴らしいですね。私にとって今回のこれが初SSDなんですが、一度体験したらSSDは堪りませんね。書き込み中に動かしてももちろん問題ないわけで快適そのものです。こうなるとMacBookもSSDにしたいです。

で、type Pの動作的な面は余計なアプリケーションを片っ端からアンインストールし不要なサービスを切ればWindows Vistaでも結構快適に動作しています。さすがにThinkPad X61sやMacBook上で動作しているものと比較すれば快適とはいきませんが、普通に満足に使える状態で動作します。肝は余計な常駐しているアプリケーションと不要なサービスですね。そんでもって、画面というか解像度ですが、やはり小さくてなかなか慣れるまでが大変ですね。もちろん1600x768という解像度は素晴らしいです！しかしやはりこのサイズには結構つらいものがあり、小さいボタンや文字をクリックするのが結構大変です。私の場合、比較的キーボードで何でも操作しているので普段の環境と同じ状態までセットアップできれば問題ないとは思いますが。

以上、昨日今日とセットアップしながら使用した感想です。これからいろいろ使っていくとまた違った感想が出てくるかもしれませんが、とにかくこのサイズでこの軽さで結構普通に使えるスペックだったtype P、今のところ私気に入っています！

映画と鍋焼きうどんと御苑


以前観た映画「CHEチェ 28歳の革命」の続きである「CHEチェ 39歳 別れの手紙」を観に行ってきました。正直なところ私はこのチェ・ゲバラなる人物について全く知らなかったのですが、前回観てカストロらとキューバ革命を起こした人物ということを理解しました。ぶっちゃけですよ、ゲリラの指導者が何故このように人気がある？のでしょうか。今ひとつ理解できないでいる私ではあります。

映画を観た後は、新宿ミロードにあるうどん鍋専門店「讃岐 みのり」で鍋焼きうどんを食してきました！鍋焼きうどん好きの私ではありますが、ここ何年と食す機会がありませんでした。しかしです、今日ついにふうふうあつあつの鍋焼きうどんを口にすることが出来たわけです。様々な種類の鍋焼きうどんがそろっていたのですが、ここはやはりオーソドックスなものをと思いそれを注文しました。ゆずの香りが漂いあっさりした味の鍋焼きうどん、これは堪りませんね。とても美味しかったです！

その後、ちょっと日が傾き始めていたのですが寒桜が満開という話なので急いで新宿御苑へチェックしに行ってきました。するとどうでしょう、満開というか散り始めている感じが出てきていました。この前観たときは咲き始めたばかりだったというのに、あっという間です。そんなことで数は少ないですが撮ってきた写真はいつも通りFlickrにアップロードしました。よろしければ観てください。

そういえば


私のVAIO type Pはいつになったら来るのでしょうか。以前は毎日のように出荷状況をチェックしていたのですが、ここ最近注文していることをすっかり忘れていました。

そんなことで思い出したようにチェックしてみたのですが未だに生産中でお届け予定日は2月下旬のままです。2ちゃんねるを見ているとそろそろ出荷早まりましたメールが来ても良いのになぁと思うのですが。。。

あ、そういえば発表されて時間が経ちましたがauの春モデルである新しいCyber-shotケータイ「S001」がめちゃくちゃ欲しいです。今私が持っているのはWILLCOM 03とdocomo SO905iとiPhoneということで、出来ればiPhoneで使用している電話番号がメインのものなのでこれをCyber-shotケータイに持って行きたいのが本音です。まあぶっちゃけ、iPhoneでメールやらいわゆる日本の携帯電話的な使い方は出来ないというか使い難くて仕方がないのですが・・・。そんなことでiPhoneをサブにして日本の携帯電話をメインにしたいです。

うむー、どうしましょう。もっともMNPでauにすれば良いだけの話なんですが、計算すると端末の残高がまだ6万円ほどあるようで、ちょっと悩んでしまう金額です。

太巻き


恵方巻きというんでしょうか、節分の日に巻き寿司を食べる習慣。元々関西地方の習慣であり、マスコミとコンビニの力により全国に広められたものと私は認識しています。今までこれを口にしたら負けだと思い続けていたのですが、ごめんなさい、負けてしまいました。

さすがに丸かぶりということは出来ず、普通に食したわけなんですが普通に美味しかったです。しかし、今日が休日だったりもう少し早く帰ってこれて時間があるような日であれば、昔々鮨を握っていた私としては太巻きを自分で作りたかったなぁと思ったりも。というか、何か鮨を見ると血が騒いできますね！巻き巻きしたいぜー。

WILLCOM新端末


そういえば昨日WILLCOMの新端末が発表されたわけなんですが、03から機種変しようかなと思っています。

最近、どちらかというと03の使用頻度が減ってしまいました。何というんでしょうか、スマートフォンは確かに便利なんですが電話機やメールをする端末として考えると普通の端末の方が断然使いやすいわけで、iPhoneも持っている私としてはスマートフォンは1台あれば十分かなと。
