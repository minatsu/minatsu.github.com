和食と映画


今日も映画を観てきました。今日のは「ザ・ムーン」です。以前何かでこの映画のことを知り絶対に観たいと思っていたもので、早速TOHOシネマズ六本木ヒルズに観に行ってきました。

でも、その前にお昼ご飯を食すため、久しぶりに六本木ヒルズを散策し、何か私好みという感じのお店が目についたのでそこで食してみることにしました。「やさい家めい」です。やさい家というだけあり野菜を中心とした和食のレストランのようで、野菜好きの私には堪りません。しかも通されたのがカウンタ席ということもありこれまたカウンタ席好きな私には堪りませんでした。あいにく時間がちょっと早めということもあり、板前さんは一人だけでしかもちょっと端っこの調理板で作業をしていたのでその様子をじっくり見ることが出来ないのが残念でした。しかしながら、私このお店が気に入りました！和食で野菜でしかも調理の様子が見られるカウンタ席、そして料理も美味しいー。六本木にはなかなか来る機会がないのですが、ここを目当てにくるというのも良いかも！

で、映画「ザ・ムーン」ですが月に行った宇宙飛行士のインタビューと当時のそれらに関する映像で構成された映画でとても面白かったです。

映画と新宿御苑


以前から気になっていた映画「感染列島」をいつもの新宿ピカデリーで観てきました。予告を観てからというものこれは観たい！とものすごく期待していたわけなんですが、うーむ・・・。つまらないということではないのですが、何といいましょうか、日本映画にこういういわゆるパニックムービーは無理というか、突っ込みどころ満載という感じでしょうか。でも、悔しいかな泣けました。突っ込みどころ満載の映画でこんなに泣いてしまうとは・・・。

そんなことで、映画を観た後は京王百貨店で催されている駅弁大会へ行って、去年同様「いわしのほっかぶり寿司」と今回初めてあじ寿司を食しました。駅弁はやはり美味しいですね！

で、おなかいっぱいになったところで久しぶりに新宿御苑に行ってきました。寒桜が咲き始めたということなのでそのチェックに。本当に咲き始めたという感じの状態でこれからが楽しみです。ちなみに、梅も一部ちょっとだけ咲き始めたという感じでした。

少ないですが撮った写真はいつものようにflickrにアップロードしました。よろしければ観てください。

VAIO Type P触ってきた


早速、朝一番に銀座にあるソニービルへ行きVAIO type Pを見て触ってきました。

2Fにはホワイト、レッド、グリーンがそれぞれあり3Fにはブラックが展示されていました。で、まず色なんですが、私的にブラックとホワイトで迷っていたのに、以外とレッドがいい感じの色していました。もっと紫っぽい赤かなと思っていたのに結構赤っぽい赤でいい感じです。悩みが増えた！

そして肝心の触り心地ですが、やはり何といっても軽い！です。で、ファンレスというだけあり左下がちょっと温かいかなという感じがしないでもなかったのですが、裏はそれなりに温かかったです。熱いというわけではないのであれですが、でもまあ展示していると言うこともありずっと電源付けっぱになっているわけなのでこの辺は実際に購入し使ってみてどうなるかという感じでしょうか。

ちょっとしかないですが撮ってきた写真はflickrにアップロードしました。

動作的な部分は、想像していたよりも普通に使える感じでした。もっさりしているのかなと思ったのですがAtom Z540であればそんなに気になりませんでした。スペック的には心配しなくてもいいかもしれません。画面のほうは、確かにこの画面サイズ解像度では文字等々小さく、クリックするのがちょっと大変かもしれないんですが、慣れの問題ではないかなと思います。たぶん。だって何たって1600x768という解像度の魅力は素晴らしいものがありますもん！

思う存分触った後、本屋さんに行ったり映画「CHEチェ 28歳の革命」を観てきました。なかなか良い映画でこれは続編を観なければなりませんね！

そんなことで、このtype P欲しい買いますよ！あとは色をどれにするか・・・。ブラックかレッド、うーむーどちらにしようかな。

VAIO Type P発表


今日発表されたVAIOの新型モバイルPC「type P」。噂されていた通りのスペックで液晶は8インチという小ささながら1600x768の解像度でCPUはAtom、しかも約650gという軽さですよ。これはもうあのバイオC1の再来といっても過言ではないのではないでしょうか。買います！

まだ実物は見てないのであれですが、写真やレビュー等々から私の中では買う！と決めています。毎度おなじみVAIO・OWNER・MADEシミュレーションでシミュレーションしてみたら\152,300になりました。えへ。しかしメインPCではない予定なので妥協してSSDを64GBにして\127,300かな。。あと悩むのは色です。はじめ見たときはホワイトがいいかなと思っていたのですが、開いた状態だとブラックがアルミMacBookと似ているので、おそろい！って感じでいいなとも思っています。

週末にでも早速実物を見に行ってきたいと思います！ちなみに今日はこのtype Pのことで頭がいっぱいでどうしようもありません！かっこいいなー。
