ThinkPad X61s


ついこの間までVAIO、VAIOと騒いでいた私でありますが、昨日ご報告したようにThinkPad X61sをポチりました。

MacBook Airに嫌気が指しWindowsマシンが欲しいと思っていたここ最近、いろいろチェックしていました。今流行の5万円PCやThinkPadやもちろんVAIO等々何かないものかと日々考えていたわけなんですが、これだ！という決め手にかけていたというか、本命としてはすでにVAIO type TZの新型を待つというのが決定事項的なもので心の中にありました。

噂では今月中に出るかもしれない的なものがあるわけで、それを待つというのが本来の流れです。しかし、もう待ちきれない？！いや、サブ的なものでもそれのつなぎになるWindowsマシンが欲しいという思いがあり、そんな今日この頃、ふと目に飛び込んできたのがこのThinkPad X61sでした。

ThinkPad Xシリーズというと、今現在LenovoのサイトでもX61が10万円を切る値段で売りに出されていますが、納期が3週間と結構手にするまで時間があり私的にはちょっと悩んでいました。しかし他のサイトで売り出されていたこのX61sはメモリー4GBで10万円を切る値段！型落ちでありHDDが80GBということもありますが、サブとしてつなぎとしても十分買いと思ったわけです。

ただ、X61シリーズの欠点としてXGAという今更この解像度はないよなというものがあるわけですが、出るであろう私の望み通りのVAIO type TZの新型を購入するまでのつなぎとして、十分使えるんじゃないかなと思うわけであります。あくまで本命はVAIO type TZの新型ですよ！まあ購入するのはもうちょっと先になりそうな感じですが。

そんなことで、ThinkPad X61sはもう出荷したそうです。受け取るのは週末になると思うのですが、明日あたり不在票が入っていそうな気配！今週末はX61sのセットアップでうはうはしてそうです。HDDは換装したいな！

新宿ピカデリー


映画を観るときはよく新宿バルト9に行きます。チケットの引き替え云々で評判的には良くない面もあったりするのですが、私的にはあの座席の座り心地と前後の間隔等々によりとても気に入っている映画館です。前の人の頭が邪魔になってスクリーンが見えないということもないし、新宿三丁目駅から近いというのも良いし。

しかし、今日はちょっぴり気分を変えて、7月に新装オープンした新宿ピカデリーに行ってきました。都心最大級のシネコンというだけありなかなか広め。チケット等のシステムはバルト9と同じでチケット売り場の長い列に並ぶかオンラインで購入し発券機でということになっています。で、今回はもちろん事前にオンラインで座席を予約していたので列に並ぶこともなくすんなりと観ることが出来ました。ちなみに、観た映画は「幸せの1ページ」です。笑いありちょっとウルウルありのなかなか良い映画でした。ただ、気になったのはこの邦題。内容的には原題である「Nim’s Island」そのままにしているほうがピッタリだと思うのですが、なんでこんな邦題を付けたんでしょうか。

そんでもって新宿ピカデリーの感想ですが、全体的には新宿バルト9と互角というか同じです。しかし、こちらのほうが良いなと思うことが一点、それはメンバーズカード、6回映画を観れば1回無料で観ることが出来るそうです。何と素晴らしい！私的お気に入りである新宿バルト9と同じであればこのメンバーズカードによる得点があるこちら新宿ピカデリーのほうがお得感があります。これからはこちらで観ようかな・・・。

D90発表


私が普段使用しているメインのカメラはNikon D80です。デジタル一眼レフということでさすがにポケットに入れてとか持ち運べるものではないので、毎日は持ち歩いていません。もっぱら週末とかそんな感じで使っているのですが、そのD80の後継機種となるD90が今日発表されました。

正直、どんなスペックなのかなとかいつ発表されるのかなとか気になり、ここ最近は毎日のようにいろいろチェックしていました。で、今日の発表、リーク情報通りのものが登場しました。ライブビュー搭載や動画撮影機能等々、噂通りの機能満載です！

私的には何といってもライブビューが魅力的です。花の写真を撮るときなどよく変な格好というか、結構無理な姿勢をとったりしているのでライブビューがものすごく魅力的ではあります。他のメーカ他の機種で搭載されているものを見るたび羨ましいと思っていたのは秘密です。

そんなことで、早く手にとって実物を触りまくりたい！今のところ買う気満々であります！

ちなみに今日の写真はOLYMPUS E-420+パンケーキレンズで撮りました。会社の新人君が実はこれのユーザであるとということを知り、ちょっと拝借し撮ってみたりしていました。小さくて軽いこの組み合わせ、実に羨ましいです。サブとして欲しいなと前々から思っていただけに、実物を触りまくり、ますます欲しくなりました。

あれもこれも、欲しいものがたくさんあって困ります。
