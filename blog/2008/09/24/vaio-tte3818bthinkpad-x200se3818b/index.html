VAIO TTかThinkPad X200sか

VAIO type TZの新型を購入するまでのつなぎとしてThinkPad X61sを購入し、日々、Windows Vistaの快適さを実感している今日この頃。そのtype TZの新型のスペックが出てきました。

TTですよTT。VAIO type TTになるのかな。TTというと車、アウディが思い浮かぶわけですが、とにかくTTです。スペック的にはtype ZからGPUを抜いて小さくした感じ、外見的には今のtype TZとどこが変わったのかあまりよくわからない感じに見えます。しかし値段がいやに高いです、この辺はもちろんカスタマイズできるとは思うのですが。。。私的にはチップセットが現行世代になりメモリーMAX4GBになったというのは大変喜ばしいです。デザインもほぼ変わらない？ようなので、これは買いかな。

しかししかし、そんなことを思っているところに今度はThinkPad X200sの販売がアメリカでは始まったようで、そのスペックがまたちょっと惹かれます。12.1インチの液晶画面でWXGA+ 、1440x900の解像度ですよ！いいな。

なんだかものすごく迷ってきました。VAIO TTとThinkPad X200s。比較すると大きさ的には279x199.8mmと295x210mmでもちろんTTのほうが小さいわけですが、20mmほどということでそれほど大きな差はないような気がします。それでいて重さは1.3kgと1.12kgということでX200sがちょっと軽い。そして解像度が1366×768と1440x900。おおおー、やばい悩みます。ただ番号が3桁になったThinkPad（もしかしたら今のところはX200シリーズだけ？）の私的欠点として、天板の隅にLenovoと彫りこまれていることです。今まで通りThinkPadのロゴだけであればよかったのですが反対の隅にメーカーロゴまで彫りこまなくてもと思うのですが。

明日になるとVAIO TTのもっと詳しい情報が出てくるのかな。ThinkPad X200sも早く日本で発表ならないかな。VAIO TTにするかThinkPad X200sにするか、それともVAIO type Zにするか、ううう、悩みます！
