VAIO Type T

今私が使用しているノートパソコンはIBM ThinkPad X40です。ACアダプタが壊れ電源を供給することが出来なくなり起動できなくなったSONY VAIO TR/2の後釜ということで使用しています。ACアダプタさえ買えば使えるハズなんですが、値段が少なからず高かったということもあり、SONY信者としては情けないことでありますがそのまま放置していたりします。

そのうちACアダプタを買おうと思い続け幾月が経ち、そんなことをしているうちにもうSONYはダメだとか思うようになったり、次買うときはMacBookにする！とかいっていた私でありますが・・・どうしたことでしょう！SONY、VAIOが欲しいと思う出来事が起こりました。

それは、今日発表されたSONY VAIO type Tです。薄くて軽くて長時間つかえてCore 2 Duoで等々、プラットホームがSanta Rosaではないのが残念ですが、それにしても久々に欲しいと思えるVAIOです。シンプルなデザインでカッコイイなぁ！

次はMacBookとかいってましたが、Macは自宅にMac miniがあるしいらないかな…。あああ、見れば見るほどこのVAIO type Tが欲しくなってきます。どうしましょう！！！
